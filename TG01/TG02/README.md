# Telegram-бот TG02 с обработкой фото, голоса и переводом

## Описание

Это адвансированный Telegram-бот на aiogram, который обрабатывает:

- **Фото**: Описание: сохраняются в папке `img` с временным меткой
- **Голосовые сообщения**: сохраняются в папке `audio` в формате OGG
- **Перевод текста**: все текстовые сообщения переводятся на английский

## требования

- Python 3.9+
- aiogram 3.0.0
- requests 2.31.0
- google-cloud-translate 3.11.1

## Установка

```bash
pip install -r requirements.txt
```

## Конфигурация

1. Откройте `config.py`
2. Вставьте токен Telegram-бота:

```python
TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"
```

3. Директории для сохранения уже конфигурируются:
   - `IMAGES_DIR = "img"`
   - `AUDIO_DIR = "audio"`

## Запуск бота

```bash
python bot.py
```

## Функционалность

### Фото
- Отправьте фото боту
- Бот автоматически сохранит его в папке `img`
- Ответ выводит с уверждением и патем к сохраненному файлу

### Голосовые сообщения
- Отправьте голосовое сообщение
- Бот сохранит в `audio` в формате .ogg
- Получите потверждение о местонахождении

### Перевод текста
- Отправьте любой текст
- Бот переведет на английский через Google Translate
- Ответ выводит оригинал и перевод
